Cara Membuat Screenshot:

1. Tekan F12 untuk buka DevTools

2. Klik icon Device Toolbar

3. Set ukuran: 1280 x 720

4. Klik 3 titik → Capture Screenshot

5. Save sebagai: screenshot-wide.png

6. Pindahkan ke folder: public/images/
